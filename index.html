Patios & Portales

Adaptación editorial en castellano de España para jugar en cuadrícula: reglas revisadas, tablas compactadas, mapas tácticos y recursos imprimibles con estética de cuaderno escolar, recreativos y tardes de los noventa.

Leer el manual Versión HTML completa, preparada para navegar desde el navegador. Fuente Markdown Texto semántico listo para exportar a PDF con Pandoc o una herramienta similar. Mapas y prompts Mapas bitmap 12×12 y prompts base para regenerarlos. Utilidades JS Generadores rápidos para crear pandillas, peques y tiradas.
