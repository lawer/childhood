Utilidades

Generadores rápidos para montar una pandilla sin estar pasando páginas: nombres, peques, atributos, trasfondos y habilidades.

Crear pandilla rápida

Genera nombre de pandilla y cuatro peques con nombre, atributos, Salud, Movimiento y Trasfondo.

Nombre de pandilla

Tira dos veces 1D100: Tabla A + Tabla B.

Nombre de peque

Tira 1D100 para nombre/apodo y 1D100 para apellido.

Atributos

Tira 1D6 para obtener la matriz y repártela entre Mente, Corazón, Cuerpo y Molar.

Trasfondo

Tira 1D100 en la tabla de Trasfondos.

Habilidad

Tira una habilidad al azar desde la lista disponible.

Tirador de dados

Atajo para tiradas comunes.

Nota técnica: estas utilidades están hechas con Alpine.js y llevan las tablas incrustadas en el propio JavaScript. Puedes abrir esta página como archivo local sin servidor.
